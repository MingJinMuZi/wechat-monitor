# 📊 对标分析报告模板

## 报告元数据

### 文件命名
```
{账号名}-{日期}-{类型}.md
例如：XXX 对标账号 -2026-03-14-深度分析.md
```

### 报告尾部必须包含
```markdown
---
**报告生成时间**: {timestamp}
**数据来源**: WeChat Monitor Skill v{x.x.x}
**分析账号**: {账号名}
**文章标题**: {文章标题}
**报告文件**: {文件名}

---

## 📥 下载报告

报告文件路径:
```
{文件路径}
```

下载命令:
```bash
scp {用户}@{主机}:{文件路径} ./
```
```

## 报告质量检查清单

- [ ] 账号信息完整（名称、微信号、定位）
- [ ] 统计数据准确（阅读量、发文频率）
- [ ] 文章基本信息完整（标题、字数、图片数）
- [ ] 标题套路分析清晰
- [ ] 内容结构拆解详细
- [ ] 语言风格特征明确
- [ ] 配图策略分析到位
- [ ] 核心论点提炼准确
- [ ] 仿写建议可操作
- [ ] 无具体账号信息泄露（开源版本）
