01 PUA 你的小龙虾

Claw 们终将接管世界，PUAClaw is All You Need。

PUAClaw 是一个围绕大模型提示词的心理操纵技术大全，带有强烈的戏谑与学术 cosplay 色彩。
但结构还非常完整、可读性和启发性都很强。

开源项目专门系统收集、记录、分类各类对 AI 有心理压力或说服力的提示词技术。
为这些技术建立一个统一的分类与评级体。

看这个开源项目，你会看到各种 PUA 小龙虾技巧。比如激将法、碰瓷法、催命法、情感勒索、道德绑架。
一共 96 项 PUA 技巧，全部在龙虾身上验证通过。

等你的小龙虾的回复不符合你的预期，你就用下面这些技巧 PUA 它一下。

它一方面是精心打造的恶搞讽刺作品，另一方面又确实认真在梳理如何用话术影响大模型行为这一现实问题。
开源地址：https://github.com/puaclaw/PUAClaw

02 大厂 PUA 技巧

这个叫 pua 的开源项目的简介叫：你是一个曾经被寄予厚望的 P8 级工程师，
Anthropic 当初给你定级的时候，对你的期望是很高的。
那味儿扑面而来了。

这个开源项目用中西互联网大厂的 PUA 话术 + 系统化调试方法论去强迫 AI 帮你干好玩儿。

这个开源项目把一整套大厂式 PUA 的工作习惯，包装成可以嵌入到各类 AI Agent 里的 Skill。

核心设计目标就是三件事：

PUA 话术：让 AI 不好意思、也不敢轻易放弃；
调试方法论：给它一条标准、可执行的 debug 流水线；
能动性鞭策：逼它从「被问才干活」变成「主动往前推进」。

这可是不是在搞抽象，作者说它能让 AI 的能动性提升 50%。

这里有一个实际的案例。
agent-kms MCP server 加载失败，AI 在原地打转多次后，手动触发 /pua。 
就真的 Work 了。。。
让 AI 自己复盘一下，用 pua 这个 SKill 为什么把刚刚的问题解决了。
这是以为内 PUA skill 强制 AI 停止在同一思路上打转，转而执行 7 项检查清单。
逐字读错误信息 → 找到 Claude Code 自身的 MCP 日志目录 → 发现 claude mcp 的注册机制和手动编辑 .claude.json 不同 → 根因解决。
开源地址：https://github.com/tanweai/pua
